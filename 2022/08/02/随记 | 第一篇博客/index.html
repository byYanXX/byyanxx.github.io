我的第一篇博客

工作流

学习：在本地 Typora 记录笔记（图片统一默认保存在博客的路径下）

图谱：知识图谱使用 obsidian 里的「白板」

博客：将想要发布的内容，拷贝到 ./source/_posts 文件中

上传：在 MyHexo 文件下执行 hexo g -d 即可自动发布到 github

一些注意的地方

  • hexo 中无法使用 ==文本== 来高亮文本，需要使用标签形式：<mark>文本</mark>

  • tags 标签全小写

  • 代码块

分类与标签

  • 分类的英文大小写敏感，标签的英文只能小写；

  • 分类为大方向，宏观的、抽象的、模糊的，不涉及具体知识体系；

  • 标签为细分小方向，要有明确的知识大领域，比如 JVM、JUC 都隶属于「Java」标签的范畴，不再单独列出 JVM、JUC 等标签；

  • 标签可能涉及多个，一般不要超过两个，比如 经验总结-[java, 备忘]、经验总结-[数据结构与算法, 归纳]。

分类 内容 标签举例
Mini-Talk xxx 工具分享、讨论、…… xxx 工具；网站收藏；……（大多时候不加标签）
经验总结 软件安装、小 bug 修复、快捷键备忘、…… 软件安装；意外修复；备忘；归纳；vim；emacs；……
计算机基础 计网、OS、数据结构…… 计网；操作系统；数据结构与算法；linux；八股；…………
源码解读 各种源码、…… java；spring；……
学习 来自网课、博客的学习笔记、…… java；k8s；jalor；spring；redis；vim；emacs；git……
随笔 创作的各种文字内容、…… 旧诗；词；现代诗；……
